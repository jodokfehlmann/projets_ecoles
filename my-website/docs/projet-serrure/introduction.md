## 🎯 Objectif

Construire un système de contrôle d'accès complet : lecteur RFID, clavier à code, afficheur LCD, servo-moteur pour le verrou et buzzer. Le tout dans un boîtier imprimé en 3D.

## 🎓 Ce que tu vas apprendre

| Compétence | Détail |
|-----------|--------|
| **RFID** | Lire les cartes et badges sans contact |
| **EEPROM** | Mémoriser les cartes autorisées |
| **Machine à états** | Structurer un programme complexe |
| **Sécurité** | Concepts d'authentification |
| **I2C** | LCD avec module I2C |

## 🔒 Fonctionnalités du système

- ✅ Accès par carte RFID
- ✅ Accès par code PIN (4 chiffres)
- ✅ Affichage LCD avec messages
- ✅ Alarme après 3 tentatives échouées
- ✅ Mode admin pour ajouter/supprimer des cartes
- ✅ LED verte (accès) / rouge (refus)
