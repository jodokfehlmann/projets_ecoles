# 🔩 Assemblage de la Serrure

**Durée estimée :** 45 minutes

## Étape 1 — Monter le mécanisme de verrou

1. Fixe le **servo SG90** dans son logement (2 vis M2)
2. Attache le **verrou rotatif** sur le bras du servo
3. Teste manuellement : le bras doit tourner de 0° à 90° librement

```
Position fermée (0°)  : ██████ ←verrou bloqué
Position ouverte (90°): ██████   ↑verrou sorti
```

## Étape 2 — Installer le lecteur RFID

1. Le **MFRC522** se fixe sur la façade du boîtier
2. L'antenne (la bobine) doit être côté extérieur
3. Distance de lecture : 3 à 5cm (cartes ISO 14443)

## Étape 3 — Monter le clavier et le LCD

1. Insère le **clavier 4×4** dans son support
2. Le LCD se place en dessous du clavier sur la façade
3. Passe les câbles à travers les fentes prévues

## Étape 4 — Finaliser le boîtier

1. Monte toutes les pièces dans le boîtier principal
2. Passe tous les câbles soigneusement
3. Les LEDs et le buzzer se montent sur la façade
4. Ferme avec la cache façade
