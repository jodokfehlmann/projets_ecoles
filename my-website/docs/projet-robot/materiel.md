# 🔧 Liste du Matériel

## Composants électroniques

| # | Composant | Qté | Rôle |
|---|-----------|-----|------|
| 1 | Arduino Uno R3 | 1 | Cerveau du robot |
| 2 | Moteur DC 6V avec réducteur | 2 | Propulsion gauche et droite |
| 3 | Driver L298N | 1 | Contrôle des 2 moteurs |
| 4 | Capteur IR TCRT5000 | 3 | Détecter la ligne noire |
| 5 | Batterie 9V ou pack 6×AA | 1 | Alimentation autonome |
| 6 | Switch ON/OFF | 1 | Allumer/éteindre |
| 7 | LED bicolore (rouge/verte) | 1 | État du robot |
| 8 | Potentiomètre 10kΩ | 1 | Réglage sensibilité |

## Pièces 3D imprimées

| Pièce | Description |
|-------|-------------|
| Châssis principal | Corps du robot (2 parties) |
| Support moteur gauche | Fixe le moteur gauche |
| Support moteur droit | Fixe le moteur droit |
| Support capteurs IR | Barre frontale porte-capteurs |
| Support batterie | Logement pile/accu |
| Roue (×2) | Diamètre 65mm |
| Roue folle | Troisième point d'appui arrière |

## Matériel de parcours

| Article | Description |
|---------|-------------|
| Ruban adhésif noir | Largeur 19mm, pour tracer le circuit |
| Sol lisse | Carrelage, vinyl ou papier blanc |

:::tip Préparer le parcours
Trace ton circuit **avant** d'assembler le robot. Utilise du ruban noir sur fond blanc ou clair. Évite les angles inférieurs à 45° pour commencer.
:::
