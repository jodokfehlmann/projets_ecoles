# 🤖 Bienvenue sur Arduino EduKit Algérie

Bienvenue dans la documentation officielle des **kits Arduino éducatifs** fabriqués localement en Algérie.

## Qu'est-ce que Arduino EduKit ?

Arduino EduKit est un ensemble de **kits pédagogiques** qui permettent aux élèves de :
- 🔧 Assembler de vrais systèmes électroniques
- 💻 Apprendre la programmation avec Arduino
- 🧠 Résoudre des problèmes réels
- 🎓 Développer des compétences en STEM

Tous les kits sont **fabriqués en Algérie** avec des pièces imprimées en 3D, et la documentation est disponible en **français et en arabe**.

## 🛠️ Ce qu'il te faut

Pour commencer, tu auras besoin de :

- Un **kit Arduino EduKit** (commandé auprès de nous)
- Un **ordinateur** avec le logiciel Arduino IDE
- Un **câble USB** (inclus dans le kit)
- De la **curiosité** ! 🎉

### Installer Arduino IDE

```bash
# Télécharge Arduino IDE 2.x sur :
# https://www.arduino.cc/en/software
```

:::tip Conseil
Télécharge **Arduino IDE 2.x** — il est beaucoup plus moderne avec de l'autocomplétion et un débogueur intégré.
:::
