# 🔩 Assemblage de l'Instrument

## Étape 1 — Préparer les électrodes touch capacitives
1. Découpe 8 morceaux de **papier aluminium** (5×5 cm chacun)
2. Replie-les pour former des pavés rigides
3. Soude ou colle avec ruban conducteur un fil sur chaque pavé
4. Place chaque pavé sous une **touche imprimée en 3D**

:::tip Alternative simple
Tu peux aussi utiliser des **vis en métal** vissées dans le boîtier comme électrodes — plus solide !
:::

## Étape 2 — Monter l'amplificateur
1. Le **PAM8403** se câble entre la sortie D3 et le haut-parleur
2. Un potentiomètre permet de régler le volume
3. Fixe le haut-parleur dans sa grille avant

## Étape 3 — Assembler le boîtier
1. Colle les électrodes sous chaque touche
2. Place les touches sur le boîtier (elles doivent se soulever légèrement)
3. Monte l'Arduino et l'ampli à l'intérieur
4. Les LEDs RGB se montent sur le dessus du boîtier
