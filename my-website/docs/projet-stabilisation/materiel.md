# 🔧 Liste du Matériel

## Version Ball-Balancing (recommandée pour débuter)
| # | Composant | Qté | Rôle |
|---|-----------|-----|------|
| 1 | Arduino Uno | 1 | Contrôleur PID temps réel |
| 2 | MPU6050 | 1 | Mesure angle en temps réel |
| 3 | Servo MG996R | 2 | Incliner le plateau (X et Y) |
| 4 | Alimentation 5V 3A | 1 | Servos haute puissance |
| 5 | Potentiomètre 10kΩ | 3 | Réglage Kp, Ki, Kd en direct |
| 6 | LCD 16×2 I2C | 1 | Afficher les paramètres PID |
| 7 | Bille en acier 25mm | 1 | Objet à équilibrer |

## Pièces 3D imprimées
| Pièce | Description |
|-------|-------------|
| Plateau carré 30×30cm | Surface sur laquelle roule la bille |
| Support servo X | Axe de rotation gauche/droite |
| Support servo Y | Axe de rotation avant/arrière |
| Bras de liaison | Relie servo au plateau |
| Base de support | Structure fixe au sol |

:::warning Sécurité
Commence TOUJOURS par la version Ball-Balancing pour maîtriser le PID avant de passer à un drone.
:::
