# 🔩 Assemblage de la Calculatrice

**Durée estimée :** 60 minutes

## Étape 1 — Préparer les touches

Les 16 touches sont imprimées en 3D. Pour une meilleure lisibilité :
1. Ponce légèrement la face supérieure de chaque touche
2. Applique du vernis blanc (ou peinture acrylique)
3. Grave les symboles avec un crayon ou stylo permanent
4. Applique une couche de vernis transparent pour protéger

```
Disposition du clavier :
┌────┬────┬────┬────┐
│ 7  │ 8  │ 9  │ ÷  │
├────┼────┼────┼────┤
│ 4  │ 5  │ 6  │ ×  │
├────┼────┼────┼────┤
│ 1  │ 2  │ 3  │ -  │
├────┼────┼────┼────┤
│ENTR│ 0  │ .  │ +  │
└────┴────┴────┴────┘
```

## Étape 2 — Assembler l'électronique

1. Soude l'Arduino Nano sur son support dans le boîtier
2. Fixe le circuit TP4056 près du port USB-C
3. Colle la batterie Li-Po dans son logement avec du velcro double-face
4. Monte l'OLED sur son support incliné à 45°

## Étape 3 — Installer le clavier

1. Pose la membrane du clavier matriciel sous la façade
2. Fais passer les fils à travers la fente prévue
3. Insère les 16 touches dans leurs logements (elles clipsent)
4. Vérifie que chaque touche a un bon retour tactile

## Étape 4 — Fermer le boîtier

1. Passe tous les câbles soigneusement
2. L'interrupteur et la LED se montent sur le côté
3. Ferme avec 4 vis M2 × 8mm

:::tip Ergonomie
Le boîtier est conçu avec une légère inclinaison arrière (5°) pour être plus confortable en main. Vérifie que l'OLED est bien lisible depuis le dessus.
:::
